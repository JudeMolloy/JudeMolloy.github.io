22nd Colour

A simple iOS game designed to confuse you.

See More

This simple iOS game, aptly named 22nd Colour gives you 20 seconds to click the colour that it says whilst the position of the coloured squares change and the colour of the text adjusts to the previous colour you clicked.

This app was one of the first iOS applications I ever built and I learned a lot about iOS development and looking back at the code with all of it's bad practices I've learned a lot of what not to do.

Unfortunately the app is currently not on the iOS App Store, however, it is likely that it will be soon and I will make sure to update this post so that it contains a link to the app.
