I built this application with the goal of becoming better at Swift - a programming language - so I could go on to build better iOS applications. This definitely helped me learn about the power of Swift and what it had to offer.

This was part of a Stanford iOS course from ITunes U that I was watching lectures from to become more familiar with Swift.

This calculator was probably the first reasonably complex iOS application that I built and it paved the way for future apps.
